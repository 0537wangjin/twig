运行环境

Twig至少需要PHP5.2.4来运行。

使用Composer安装

先创建一个 composer.json 下载

{
    "require": {
        "twig/twig": "1.29.0",
        "php": ">=5.2.7"
    },
    "repositories": {
        "packagist": {
            "type": "composer",
            "url": "https://packagist.phpcomposer.com"
        }
    }
}

开始安装

composer install

开始安装

如果你不使用Composer，你可以点击下载代码文件
