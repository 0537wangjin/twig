with

使用with创建一个内部作用域，值在范围之外不变：

{% verbatim %} 
{% set foo = 99 %}
>>头部foo的值是 {{ foo }}<<
{% with %}
    {% set foo = 42 %}
    foo is {{ foo }} here
{% endwith %}
>>尾部foo的值是 {{ foo }}<<
{% endverbatim %}

输出结果： >>头部foo的值是99<< foo is 42 here >>尾部foo的值是99<<
